Vue历练之路一---Vue介绍、安装及项目初始化

认识Vue

​ Vue(读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动。

渐进式框架

​ 渐进式框架就是主张最少，它具有比较强的包容性。例如一个项目使用的是Jquery的框架，在使用Vue升级时，不必一次性将所有已有的项目全部重做，只需要自底向上慢慢去替换，就可以完成迭代工作。

MVVM模型

​ MVVM是Model（M，数据层），View（V，视图层），ViewModel（VM，业务逻辑层）的统称。MVVM 将数据双向绑定（data-binding）作为核心思想，View 和 Model 之间没有联系，它们通过 ViewModel 这个桥梁进行交互。数据层与视图层之间的交互是相互的。数据的变化会自动同步到视图中，反之，视图的变化也会自动同步到数据上。

Vue的安装

Vue 2.x和Vue 3.x`安装方式较为不同。

Vue 2.x版本安装

npm install vue-cli -g  -- 全局安装Vue 2.x
vue --version           -- 查看Vue安装版本,出现版本提示则表示安装成功

Vue 2.x项目创建

-- 创建项目
vue init webpack 项目名称
-- 出现如下提示
Project name (项目)                     -- 输入项目名称 回车默认为vue init 时的项目名
Project description A Vue.js project    -- 输入项目描述
Author<>                                -- 输入作者
Vue build standalone                    -- 需要不需要安装编译器，我选择需要安装
Install vue-router? Yes                 -- 是否使用vue-router Y为使用N不使用
Use ESLint to lint your code? (Y/n)     -- 是否使用ESLint语法检查 
Set up unit tests No                    -- 是否使用unit测试 我没有用到选的是不选择
Setup e2e tests with Nightwatch? No     -- 是否使用e2e测试 我也没有用到选择的不使用

-- 启动项目
cd 项目
npm run dev

Vue 3.x版本安装

npm install -g @vue/cli  -- 全局安装Vue 3.x
vue --version            -- 查看Vue安装版本,出现版本提示则表示安装成功

Vue 3.x项目创建

-- 创建项目
vue create 项目名称
-- 出现如下提示
Please pick a preset: Manually select features
Check the features needed for your project: Babel, Router, Vuex, CSS Pre-processors, Linter
Use history mode for router? (Requires proper server setup for index fallback in production) No
Pick a CSS pre-processor (PostCSS, Autoprefixer and CSS Modules are supported by default): Less
Pick a linter / formatter config: Prettier
Pick additional lint features: Lint on save
Where do you prefer placing config for Babel, ESLint, etc.? In package.json
Save this as a preset for future projects? (y/N) n

-- 启动项目
cd 项目
npm run dev

​ Vue 3.x提供了图形化界面的项目搭建方式，可以使用vue ui命令进入vue管理界面进行项目的创建。
